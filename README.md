# RI3_ClassesComAtributosPrivados
Projeto de POO em JavaScript (RI 3). Foca no uso de classes com atributos privados, encapsulamento (get), e o uso de coleções (Set) para gerenciar múltiplos telefones e clientes. Implementa as classes Empresa, Cliente, Telefone e Endereco, gerando uma descrição formatada detalhada da empresa e sua lista de clientes.
